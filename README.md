# 💿 MediaShoreProject — Автоматизированная информационная система (АИС)

**MediaShoreProject** — это современное Desktop-приложение для автоматизации работы магазина медиа-носителей (дисков). Проект разработан как часть дипломной работы по специальности «Программная инженерия» и ориентирован на эффективное управление складскими запасами и заказами пользователей.

---

## 🛠 Технический стек

Система базируется на стеке технологий Microsoft, что обеспечивает высокую производительность и надежность в Windows-среде:

*   **Ядро:** C# (.NET Framework 4.8).
*   **Интерфейс:** WPF (Windows Presentation Foundation) с использованием XAML-разметки для создания отзывчивого UI.
*   **База данных:** Microsoft SQL Server 2022.
*   **ORM:** ADO.NET Entity Data Model (Database First), обеспечивающий типизированное взаимодействие с БД.

---

## 🏗 Архитектура и функционал

### 🛒 Логика корзины и складского учета
В приложении реализована сложная система синхронизации «Витрина — Склад — Корзина»:
*   **Динамический учет:** При добавлении товара в корзину (`+`) поле `discQuantityInStock` в таблице `Discs` мгновенно уменьшается, а при удалении (`-`) — возвращается на склад.
*   **Состояние заказа:** Используется система статусов (ID 4 — «В корзине», ID 3 — «Оплачен»). Это позволяет сохранять черновики заказов для каждого пользователя.
*   **Фиксация цены:** При добавлении диска в `OrderItem` записывается `PriceAtTime`, что гарантирует неизменность стоимости для клиента после оформления.

### 🔒 Безопасность и роли
*   **Авторизация:** Разграничение доступа на основе ролей (Admin/User).
*   **Финансовый контроль:** Интегрированная проверка баланса пользователя (`userBalance`). Система блокирует оформление заказа при нехватке средств.

## 📊 Структура базы данных (`DIPLOMAT_01`)

Проект использует реляционную модель данных, оптимизированную под требования ГОСТ 19.508-79.

| Таблица | Описание |
| :--- | :--- |
| `Users` | Профили пользователей, пароли, роли и финансовый баланс. |
| `Discs` | Основной каталог носителей (названия, авторы, остатки, пути к обложкам). |
| `Orders` | Заголовочная информация о заказах и их текущем статусе. |
| `OrderItem` | Состав заказов, связывающий диски с конкретными транзакциями. |
| `Statuses` | Справочник состояний заказа (Корзина, Оплачен, Доставляется). |

## 🎨 Дизайн-концепция
Приложение следует **Retro-Minimalist** эстетике:
*   Строгая сеточная разметка (`Grid`) для легкого сканирования данных.
*   Цветовая палитра: синий, черный, белый с яркими акцентами на функциональных кнопках.
*   Визуализация: отображение обложек дисков напрямую из файловой системы через привязку путей (`discCoverPath`).

## 💻 Установка и запуск
1.  Разверните базу данных, используя SQL-скрипты прикрепленные к проекту.
2.  Откройте файл решения `.sln` в **Visual Studio 2022**.
3.  Настройте строку подключения в `App.config` под ваш экземпляр SQL Server.
